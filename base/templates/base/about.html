About the Project

  • Made with Django - the Python Web Framework

    This Project is made with Django and Django templates, utilizing python for the backend code and HTML, CSS & JavaScript for the frontend code.

  • YFinance API

    It makes use of Yahoo Finance API through the yFinance library. It fetches data using the library, stores it in a django model, and fetches the data on the frontend and shows a BarGraph Visualization using D3.js

  • More Stocks Coming Soon

    Click on the individual stocks in the Top Navigation Bar's StockData dropdown to access the last 1 month data of the 5 selected stocks
